[toc]

###  帮助启动类命令

1. 启动docker

   ```
   systemcl start docker
   ```

2. 停止docker

   ```
   systemctl stop docker
   ```

3. 重启docker

   ```
   systemctl restart docker
   ```

4. 查看docker状态

   ```
   systemctl status docker
   ```

5. 开机启动（没有配置的话，每次都要手动开机）

   ```
   systemctl enable docker
   ```

6. 查看docker概要信息

   ```
   docker info
   ```

7. 查看docker整体帮助文档

   ```
   docker --help
   ```

8. 查看docker命令帮助文档

   ```
   docker 具体命令 --help
   ```



**统计容器信息–参考Linux统计磁盘**

+ Linux

```
df -h
```

+ Docker

```
docker system df
```
